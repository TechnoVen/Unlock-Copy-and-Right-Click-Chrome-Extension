
# Unlock Copy & Right-Click

This Chrome extension empowers you to re-enable copy, paste, and right-click functionality on websites that try to restrict these actions. Take back control of your browsing experience!

## Features

*   **Per-Site Control:** Activate or deactivate the extension for individual websites with a simple toggle switch in the extension's popup.
*   **Re-enables Right-Click & Clipboard:** Automatically prevents websites from disabling the context menu (right-click) and blocking copy, cut, and paste actions.
*   **Clear Visual Feedback:** The extension icon changes color to provide an immediate visual cue, letting you know whether it's active on the current webpage.
*   **Easy Configuration:** A dedicated options page lists all the websites you've enabled the extension on, allowing you to easily manage and remove sites from your list.

## How to Use

1.  **Activate on a Website:** When you visit a website that blocks copying or right-clicking, click the extension's icon in your browser toolbar. A popup will appear.
2.  **Toggle the Switch:** Click the toggle switch to enable the extension for that specific website. The extension icon will change to its "active" color.
3.  **Manage Enabled Sites:**
    *   Right-click the extension icon and select "Options".
    *   On the options page, you will see a list of all the websites where the extension is currently enabled.
    *   Click the "Remove" button next to any website to disable the extension for that site and remove it from your list.
